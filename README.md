# react-native-swipe-deck
A swipe deck for react native in simular style as tinder
## Installation

```sh
npm install react-native-swipe-deck
```

## Usage

```js
import { SwipeDeckView } from "react-native-swipe-deck";

// ...

<SwipeDeckView color="tomato" />
```
